This feature will improve the user’s productivity and also the user’s experience with the application. The goal of this feature is to take the current landing page, the watchlist, and change that default value to the landing page. This feature, the landing page, will allow for the user upon opening the application to view information with regards to mango, the sensors in use, and also potentially the watchlist interface. Once the user leaves the landing page they can also make use of the designed icon in the navigation bar to go back to the landing page.
